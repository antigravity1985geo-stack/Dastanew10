## 4. პრიორიტეტული განვითარების Roadmap

რეკომენდებული თანამიმდევრობა, ბიზნეს ღირებულებისა და კონკურენტული აუცილებლობის მიხედვით:

### ფაზა 1 — „აუცილებელი მინიმუმი" (1-2 თვე)

```
📦 1. პროდუქტის უკან დაბრუნება (Returns)
   - POS-ში return flow
   - მარაგის ავტომატური აღდგენა
   - ბუღალტრული reversal entry
   - RS.GE-ზე credit note

🏷️ 2. ფასების ეტიკეტების ბეჭდვა
   - პროდუქტისთვის label გენერაცია
   - ბარკოდი + ფასი + სახელი
   - Batch print (მრავალი პროდუქტი)
   - თერმული პრინტერის მხარდაჭერა

📋 3. ინვენტარიზაცია (Stocktaking)
   - აღწერის სესიის შექმნა
   - მარაგის დათვლა (scan/manual)
   - Plan vs Actual შედარება
   - სხვაობის ჩამოწერა/დამატება
```

### ფაზა 2 — „კონკურენტული პარიტეტი" (2-3 თვე)

```
🏭 4. წარმოების მოდული (Production/Recipe)
   - რეცეპტის/ტექნოლოგიური ბარათის შექმნა
   - ნედლეული → მზა პროდუქტი
   - ავტომატური მარაგის კორექტირება
   - წარმოების ხარჯთაღრიცხვა (COGS)

💰 5. ხელფასების მოდული (Payroll)
   - Gross → Net გამოთვლა
   - 20% საშემოსავლო, 2%+2% საპენსიო
   - დაზღვევა, კვება, ბონუსები
   - RS.GE ავტომატური ატვირთვა
   - ხელფასის უწყისის PDF/Excel

🏢 6. ფილიალების მართვა
   - Multi-branch warehouse
   - ფილიალებს შორის ტრანსფერი
   - ფილიალ-სპეციფიკური რეპორტები
   - ცენტრალიზებული ფასების მართვა
```

### ფაზა 3 — „ბაზრის ლიდერობა" (3-6 თვე)

```
❤️ 7. ლოიალობის პროგრამა
   - ქულების დაგროვება გაყიდვებზე
   - ბონუს ბარათები / ფასდაკლების ბარათები
   - სასაჩუქრე ბარათები (Gift Cards)
   - მომხმარებლის ისტორია და სეგმენტაცია

📅 8. ვადის ამოწურვის თრექინგი (Expiry/FEFO)
   - პროდუქტზე ვადის ველი
   - FEFO (First-Expired-First-Out) რეკომენდაცია
   - Dashboard alert: "10 პროდუქტს ვადა ეწურება 7 დღეში"
   - ავტომატური ჩამოწერის workflow

🛒 9. E-Commerce ინტეგრაცია
   - Glovo / Wolt / Bolt API
   - WooCommerce Sync
   - Extra.ge / Vendoo კატალოგი
   - ერთიანი Order Management

📋 10. Purchase Orders
   - PO-ს შექმნა მომწოდებლისთვის
   - PO → მიღება → Invoice matching
   - Auto-reorder (minStockLevel-ზე)
   - მომწოდებლის history და rating
```

## 5. დასკვნა

### DASTA-ს ძლიერი მხარეები კონკურენტებთან შედარებით:
- 🤖 **AI ინტეგრაცია** — ერთადერთი ქართული ERP AI ასისტენტით
- 🎨 **თანამედროვე UI/UX** — Aurora Design ≫ კონკურენტების მოძველებული ინტერფეისი
- ⚡ **Real-time Sync** — Supabase channels ≫ manual refresh
- 📱 **PWA/Offline** — ბრაუზერშივე მუშაობა, არ სჭირდება ინსტალაცია
- 🔒 **უსაფრთხოება** — RLS + PIN + Audit ≫ კონკურენტების ძირითადი auth

### DASTA-ს სისუსტეები:
- ❌ **არ აქვს Payroll** — Apex/Fina/Oris-ს აქვს
- ❌ **არ აქვს Production** — Apex/Fina/Optimo-ს აქვს
- ❌ **არ აქვს Returns** — ყველა კონკურენტს აქვს
- ❌ **არ აქვს Inventory Count** — ყველა კონკურენტს აქვს
- ❌ **არ აქვს Branch Management** — Apex/Fina/Optimo/Oris-ს აქვს

### რეკომენდაცია:
> **ფაზა 1 (Returns + Price Tags + Inventory)** ყველაზე სწრაფად დახურავს კრიტიკულ gap-ებს. ეს სამივე ფუნქცია **ყველა** კონკურენტს აქვს და მათ გარეშე მომხმარებლები ვერ გადმოვლენ კონკურენტებიდან.
